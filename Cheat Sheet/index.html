HTML Elements Attributes CSS BASH
HTML Elements
<head> Gives the browser information about the page itself, does not appear in the text.
<title> Usually nested under the <head> element, it will title your document in the "site title".
<body> Only content inside this element will be displayed on the screen.
<h1-h6> Used for headings with h1 being the largest and h6 being the smallest.
<div> Stands for (Division). It's a container that divides the page into sections, grouping elements together and are useful for applying custom styles to it specifically. *must be used sparingly*
<a> "Anchor" will separate a part of the text that you can turn into a link by utilizing the href= attribute.
<p> "Paragraph" contain a block of plain text.
<span> To be followed after <p> separates small pieces of text on the same line of other text.
<br> "Line break," *does not use a closing tag* and will manually break your sentence to the next line of the page. Can be used to backdoor paragraphs in one <p>.
<ul> "Unordered list" will list items in order using bullet points.
<ol> "Ordered list" will list items in order using numbers instead of bullet points.
<li> "List item" to nested within <ul> or <ol> to create your list.
<img/> Is a self closing tag that will add an image, but must contain a source code within Ex. <img src="image-location.jpg/>.
<video> A lot like <img> element but requires a closing tag, instead of alt= you can put it in between opening and closing tag.
<table> Creates a table but requires rows.
<tr> (Table row) to be nested within <table> to create rows.
<td> (Table data) to be nested within <tr> to create the content within the rows, you can add mutliple <tr>'s to create additional columns.
<th> (Table head) can be added to create a name for the row, or can name columns if formatted like a row but the first needs to be blank to create a blank corner cell *link with scope= "col" OR "row" to specify if you're naming a column or a row.
<tbody> Like <body> for html but specifies for table.
<thead> Like <head> for html but creates the top row as a header.
<tfoot> Specifies the end row of a table.
Attributes
<id> Ex. <div id="intro"> gives a specific name to an element.
<width> Width="xxx"
<height> Height="xxx"
<controls> Will add specific controls to things like videos.
<src> Or "source" will link your page to to another page or file, usually used for images or videos.
<href> Hyperlink reference, like src it will link to another webpage.
<target> select where a link will open, target="_blank" will open in a new window.
<#id> Will link to an id in the same webpage Ex. href="#top".
<scope="row" OR "col"> Specify if you're naming rows or columns.
<colspan="x"> Will span a blank box across x number of column cells in tables.
<rowspan> Same as above but for rows.
<em> Emphasizes text (italic).
<strong> Highlights important text (bold)
CSS
<link href='style.css' rel='stylesheet'>
Colors Color: Blue OR Hex:#000000 OR Rgb(143, 69, 18) OR Hsl(hue,saturation, brightness) hsl(120, 60%, 0.1), using hsla(34, 100%, 50%, 0.1) will add alpha a.k.a. OPACITY the same goes for rgba as well, and even hexadecimal by adding 2 additional numbers at the end instead of the base 6.
Font-family Changes the font of the selected element(multi words needs to be surrounded with ' ') *look for web safe fonts* fallback fonts can be chained with commas. Don't forget about Serif/Sans-serif
Font-weight Bold, normal, lighter, bolder. You can also use numbers 400=Normal and 700=Bold
Font-style Italic, normal.
Text-transform Uppercase, lowercase.
Letter/word spacing Determines how far apart each letter is in pixles.
Line-height Sets the line height from the bottom of the line.
Text-align Aligns text to its parent element (left, right etc.
@font-face Will allow you to use downloaded fonts, you then have to use font-family and make a name in quotations, followed by src: url(fonts/name.woff2 or .woff or .ttf format) Woff2, woff, Truetype.
BASH (Command Prompt)
$ Is the original shell basically stating it's ready for a command.
ls (list) will list all docs under your directory.
pwd (Print working directory) will show the exact path to the current working directory that you're in.
cd (Change directory) will change your directory to the targeted location.
../ or ../../ Will show/change you to a previous or further directory, this chain will even work with specific directories (a/b).
mkdir (make directory "xxx") will create a new directory with "xxx" name.
touch will create a new file inside of a directory (keyboard.txt).
clear Will clear your command feed but won't undo or delete your commands made.
